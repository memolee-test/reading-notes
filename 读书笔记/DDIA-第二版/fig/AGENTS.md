# fig — DDIA 原书插图

## Purpose
存放 DDIA 第二版原书的插图 PNG 文件，供章节笔记中通过 `![[fig/ddia_CCNN.png]]` 语法嵌入引用。

## Key Files

| File | Description |
|------|-------------|
| `ddia_CCNN.png` | 插图文件，CC=章节号(01-14)，NN=该章图片序号 |

## Subdirectories

无子目录。

## For AI Agents

### Working In This Directory
- 插图文件为静态资源，无需修改
- 文件名严格遵循 `ddia_CCNN.png` 格式，新增插图时请保持一致
- 图片通过 Obsidian 的 `![[fig/filename.png]]` 嵌入语法在章节笔记中引用

### Common Patterns
- 每章插图数量不等（如第6章 16 张，第1章仅 2 张）
- 第13-14章为第二版新增内容，插图较少
